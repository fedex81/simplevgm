Simple VGM
----------
a (very) simple - Java based - [VGM][1] 1.50 player  

Supported chips:
- TI SN76489
- Yamaha YM2612
- Yamaha YM2413

How To
-----
Download the latest jar file from the releases area  
and run from the command line:  
`java -jar simplevgm-19.0117.jar <file>|<folder>`
